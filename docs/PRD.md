# PRD: PDF Merge Service

## 목표
여러 PDF를 웹 UI 또는 터미널에서 빠르게 하나의 PDF로 병합하고 바로 다운로드할 수 있는 무인증 도구를 만든다.

## 사용자
- 로그인 없이 단발성으로 PDF를 병합하려는 일반 사용자
- 반복 병합 작업을 터미널 명령으로 처리하려는 개발자 또는 운영자

## 핵심 기능
1. 웹 UI에서 여러 PDF를 업로드하고 목록으로 확인할 수 있어야 한다.
2. 업로드된 PDF의 순서를 드래그 앤 드롭 또는 위/아래 이동 버튼으로 조정할 수 있어야 한다.
3. 서버에서 PDF를 병합하고, 진행 상태를 단계형으로 시각화해야 한다.
4. 병합이 완료되면 자동 다운로드를 시도하고, 실패하거나 놓친 경우 재다운로드 버튼을 제공해야 한다.
5. 터미널에서 로컬 파일 경로를 입력해 동일한 규칙으로 PDF를 병합할 수 있어야 한다.

## MVP 제외 사항
- 로그인, 사용자 계정, 작업 이력, 영구 저장
- PDF 분할, 회전, 압축, 미리보기, 페이지 단위 편집
- 암호 해제, 암호 입력 UI, 손상 PDF 복구
- 북마크, 폼 필드, 첨부파일, 원본 메타데이터 보존
- 비동기 작업 큐, 작업 재시도 큐, 전역 npm 배포

## 디자인
- 라이트 모드 고정
- 친근한 소비자 앱형 생산성 도구
- 중간 밀도 레이아웃
- 웜 화이트 배경 + 순백 카드
- 블루 포인트 컬러
- 스텝 바 + 상태 카드 기반 진행 상태 표현
- 모바일에서는 1열 작업 흐름으로 재배치
